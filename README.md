# SWE Match: Brain Controlled RC Car

This project uses a (k-NN *Change later) Logistic Regression model to identify commands of forward, backward, and stop based on EMG (muscle activity) from your arm to control a robotic car.

## Requirements

- Python 3.9+
- Required packages:
  ```bash
  pip install pandas scikit-learn joblib

## Usage

Run the script using:

```
python main.py
```
