# SWE Match: Brain Controlled RC Car

This project uses a k-NN model to identify commands of forward, backward, and stop based on EMG or muscle activity from your arm to control a robotic car.

## Requirements

- Python 3.9+
- Required packages:
  ```bash
  pip install
  ```

## Usage

Run the script using:

```
python main.py
```
